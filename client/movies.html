A. how to start server

  1. create a database in mysql
    mysql -u root -p123456
    CREATE DATABASE movie_db;
    quit
  2. import the database
    mysql -u root -p123456 movie_db < movie_db.sql
  3. start the server
    npm install
    npx prisma generate
    npx nodemon app.js

B. how to Use Client

  1. browse film
    use the page size and page number to browse the film, click go to refresh the list
    use the search bar to search by title or id, result will be shown in the list
    whenever you want to fresh, click the GO
    click the arrow beside the title to see the detail of the film, you can fold it later
  2. edit or add movie
    you can also add a new movie by clicking the add movie button, fill in the form and click the add button
    when you in the movie detail page, you can edit the movie by clicking the edit button, cancel the edit by clicking the cancel button
    id and genres are not editable now
  3. delete movie
    you can delete the movie by clicking the delete button in the movie detail page
